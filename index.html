HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta elements contains page information used by the browser.
  • The parent element contains multiple child elements within the body. They describe the body of the html.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A declaration contains two components. The CSS property and the value assigned to it.
  • DRY is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git clone [url]: downloads a project with the entire history from the remote repository.
  • git add: adds files to staging area.
  • git checkout main: discards changes in working directory
  • git pull: fetch from and integrate with another repository or a local branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Control flow use if statements to determine the next step
  • An array is used to test multiple data points or values stored in a variable at a time
  • An index is number assigned to each item in an array
  • Index numbers start with 0 and increase by 1 whole number for each data point.
  • A funtion is set of instructions that tells the computer how to complete a task
